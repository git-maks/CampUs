# CampUs PWA Demo

This is a frontend-only, mobile-first Progressive Web App (PWA) built specifically for a demo presentation for Erasmus students in Poland (pinned to Lublin University of Technology).

## Features
- **PWA Ready**: Works offline with Service Workers and is installable.
- **Glassmorphism UI**: "Apple-style" product design using Tailwind CSS.
- **Dummy Data Included**: All data uses local JSON records.
- **Dual Currency System**: Prices are simultaneously shown in PLN and EUR.
- **Mock Features Check**: Housing filters, interactive navigation, and simulated wallets.

## Getting Started
It's just standard Vite React!

1. Make sure you are in the project folder:
   ```bash
   cd CampUs
   ```
2. Install dependencies (if you haven't):
   ```bash
   npm install
   ```
3. Run the development server and scan the local network IP or open the localhost preview:
   ```bash
   npm run dev
   ```

## Development info
Created automatically with Vite, React, Tailwind, and React Router.
